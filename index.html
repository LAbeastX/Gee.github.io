Automation Testing – AutomationExercise.com

Performed manual and automated testing on the AutomationExercise.com e-commerce site using Selenium and PyTest. Created detailed test cases, executed regression testing, and logged bugs.

Sample Selenium + PyTest Script:

import pytest
from selenium import webdriver

@pytest.fixture
def driver():
    driver = webdriver.Chrome()
    yield driver
    driver.quit()

def test_homepage_title(driver):
    driver.get("https://automationexercise.com")
    assert "Automation Exercise" in driver.title

How to run:

1. Install Python and pip
2. Install Selenium and PyTest: pip install selenium pytest
3. Download ChromeDriver and place it in your PATH
4. Save this script as test_automationexercise.py
5. Run tests: pytest test_automationexercise.py
